#### 問題 1
ある企業が、複数のアベイラビリティーゾーンにまたがるWebアプリケーションを構築しています。アプリケーションは、パブリックサブネットにApplication Load Balancer（ALB）、プライベートサブネットにWebサーバー、さらに分離されたプライベートサブネットにデータベースサーバーを配置する3層アーキテクチャを採用しています。セキュリティを最大化するため、データベースサーバーはWebサーバーからのみアクセス可能にし、インターネットからは完全に遮断したいと考えています。この要件を満たすVPC設計として最適なものはどれですか？
- A パブリックサブネット、プライベートサブネット、データベースサブネットを作成し、データベースサブネットにはNATゲートウェイを配置しない
- B パブリックサブネット、プライベートサブネット、データベースサブネットを作成し、すべてのサブネットにNATゲートウェイを配置する
- C パブリックサブネットとプライベートサブネットのみを作成し、データベースサーバーもプライベートサブネットに配置する
- D すべてのサーバーをパブリックサブネットに配置し、セキュリティグループでアクセス制御を行う
