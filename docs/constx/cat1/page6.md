# 第6項：整理・整頓・清潔の保持（4S/5S活動）

## 1. はじめに：整理整頓の必要性

建設現場における労働災害の多くは、「つまずき」「転倒」「資材の崩壊」など、現場が散らかっていることが直接の原因で起きています。

**整理整頓は、仕事のついでではなく、仕事そのものである**  
この意識を持つことが、安全な現場作りの第一歩です。

---

## 2. 5Sの正しい定義

「5S」とは何か、それぞれの言葉の正確な意味を理解しましょう 。

| 用語 | 意味とアクション | 目的 |
| --- | --- | --- |
| **整理** (Seiri) | 必要なものと不要なものを区別し、**不要なものを捨てる**こと 。 | スペースの確保、無駄の排除 |
| **整頓** (Seiton) | 決められたものを決められた場所に置き、**いつでも取り出せる状態**にしておくこと 。 | 探す時間の削減、緊急時の対応 |
| **清掃** (Seiso) | 常に清掃をして、職場を**清潔に保つ**こと 。 | 異常の発見、環境維持 |
| **清潔** (Seiketsu) | 3S（整理・整頓・清掃）を**維持する**こと 。 | 衛生状態の確保 |
| **躾** (Shitsuke) | 決められたルール・手順を**正しく守る習慣**をつけること 。 | 安全意識の定着 |

:::tip **Point:**
「整頓」とは、ただ綺麗に並べることではなく、「必要な時にすぐに使える状態（定位置管理）」にすることです。
:::
---

## 3. 「整理整頓不足」が引き起こす事故

現場が散らかっていると、どのような災害が発生するのでしょうか。


### 3-1. 転倒災害（つまずき・滑り）

* **通路の障害物**: 通路や階段、足場の上に資材や工具を放置すると通行の妨げになり、つまずいて転倒したり緊急時の避難が遅れたりします 。
* **水濡れ・油汚れ**: 床に水たまりや油、砂がこぼれていると、滑って転倒する原因になります。  
発見次第、速やかに清掃が必要です 。

### 3-2. 飛来・落下災害

* **足場上の放置**: 足場の上に不用意に物を置くと振動や接触で下に落下し、下の作業員を直撃する恐れがあります 。
* **開口部**: 開口部の養生（蓋など）が整理されておらず外れていると人や物が落下します。

### 3-3. 火災・緊急時のリスク

* **消火設備の遮断**: 消火器、消火栓、分電盤の前に物を置くと、いざという時に使用できず、被害が拡大します 。
* **可燃物の放置**: 塗料や段ボールなどの可燃物が整理されず放置されていると、溶接火花などで引火するリスクが高まります。

## 4. 施工管理者が徹底すべき「現場の整理整頓ルール」

施工管理者は、作業員に対して以下のルールを遵守させるよう指導・監督します。

### 4-1. 通路と足場の確保

* **通路の確保**: 通路、階段、昇降口には絶対に物を置かせないでください 。
* **足場の整理**: 足場上には物を置かないことが原則です。どうしても置く必要がある場合は、落下防止措置（固縛など）を行わせます 。

### 4-2. 資材・工具の管理

* **定位置管理**: 工具や資材は「置き場所」を決め、使用後は必ず元の場所に戻させます 。
* **積上げルール**: 資材を積み重ねる際は、崩壊しないよう直角・平行に安定させ、原則として高さを**2m以下**に制限します 。
* **転がり防止**: パイプなど丸いものは、転がり止め（歯止め）の措置を講じます 。

### 4-3. 廃棄物の処理

* **都度処理**: 廃棄物（ゴミ）は、作業終了後だけでなく、**一仕事が終わる都度**分別して処分させます 。
* **分別の徹底**: 産業廃棄物は種類ごとに分別しなければなりません。

---

## 5. まとめ

現場が整理整頓されているかどうかは、その現場の「安全レベル」を映す鏡です。  
汚い現場では、必ずと言っていいほど事故が起きます。

**施工管理者のチェックリスト:**  
    1. **通路**に物が置かれていないか？
    2. **消火器**の前が塞がれていないか？
    3. **足場**の上に不要な部材が残っていないか？
    4. **ゴミ**が分別されず散乱していないか？

「片付けなさい」と指示するだけでなく、**なぜ片付けが必要か**（命を守るため）を伝え、安全で快適な職場環境を作りましょう 。
