# 建設会社について

<iframe src="https://drive.google.com/file/d/126E-p_fKo2oNcLXfo2P2l3m35MdJphuj/preview" width="640" height="360" allow="fullscreen"></iframe>

---

日本には建設会社が数多くあります。  
TVCMや野球場の広告、建設現場の囲いに書いてある会社などは全て建設会社です。  
日本の建設会社は2023年時点で約479,000社に上り、それぞれ役割に応じた階層構造を形成しています。

## ゼネコン(総合請負業者)  

---

正式名称は、ゼネラルコンストラクターになります。
元請け業者として、建設工事や土木工事を一式で発注者から直接請け負う建設会社のことです。  
建設現場では、現場で起こった全てのことに責任を持ち、下請け業者への指揮・命令を行います。

:::tip
**スーパーゼネコン**  
スーパーゼネコンとは、日本の建設業界において売上高、技術力、施工実績のすべてにおいてトップクラスを誇る大手総合建設会社5社の総称です。  
明確な法律上の定義はありませんが、一般的に「単体売上高が1兆円を超えること」が大きな目安とされています。  

| 企業名 | 主な特徴・強み | 代表的な施工実績（例） |
| :--- | :--- | :--- |
| **大林組** | 国内外に広く事業を展開し、高い技術力を誇ります。 | 東京スカイツリー、虎ノ門ヒルズ、ドバイ・メトロなど |
| **鹿島建設** | 「土木の鹿島」と呼ばれるほど土木分野に強く、超高層ビルでも業界をリードしています。 | 明石海峡大橋（JV）、フジテレビ本社ビル、ダム・トンネル多数 |
| **大成建設** | 「地図に残る仕事。」を掲げ、非同族経営ならではの自由な社風で大規模案件に強みを持ちます。 | 新国立競技場（JV）、横浜ランドマークタワー、都庁第一本庁舎 |
| **清水建設** | 1804年創業。宮大工の流れを汲み、伝統建築から最先端の工場・病院建設まで幅広く手掛けます。 | 歌舞伎座（再開発）、サクラマチ クマモト、大規模病院施設 |
| **竹中工務店** | 5社で唯一の非上場かつ「建築専業」に近いスタイル。意匠（デザイン）と施工の品質が極めて高いのが特徴です。 | 東京ドーム、あべのハルカス、日本武道館 |

:::

## サブコン(専門工事業者)  

---

正式名称はサブコンストラクターになります。
ゼネコンは現場全体を管理していく建設会社なのに対し、サブコンは設備工事や杭工事などの工事の一部を専門的に請け負う建設会社のことを指します。
職人さんの会社も専門的に工事を行う為サブコンにあたりますが、実際に施工を行わず、管理のみを行う専門工事業者をサブコンと呼ぶことが多いです。

## JV(ジョイントベンチャー/共同企業体)

---

JVとは、ジョイントベンチャー(共同企業体)の頭文字をとって略したもので、複数の企業が力を合わせて計画・施工を行うことを指します。  
建設会社(ゼネコン)が単独で受注及び施工を行う通常の場合と異なり、複数の建設会社が1つの工事を受注及び施工することを目的として形成する事業組織体のことを言います。  
建設プロジェクトは規模が大きかったり、技術的に難しかったりすることが多いです。  
1社だけでプロジェクトを進めるのが難しい場合、複数の会社がそれぞれの得意分野やリソースを持ち寄ることで、プロジェクトを成功させる可能性が高まります。

JVのメリット・デメリットについて以下の表にまとめてあります。

| メリット | 具体的なメリット | 事例・シチュエーション |
| :--- | :--- | :--- |
| **技術・強み** | **各社の強みを活かせる** | **【橋梁とトンネルの複合工事】**<br />橋に強いA社とトンネルに強いB社が組むことで、最高水準の技術を投入し品質を高める。 |
| **リスク管理** | **リスク分散ができる** | **【長期・数千億円規模の工事】**<br />不測の事態（災害や資材高騰）による赤字を1社で抱えず分担し、連鎖倒産を防ぐ。 |
| **対応力** | **大規模プロジェクトに対応できる** | **【国家規模のインフラ整備】**<br />1社では人手や重機が足りない巨大ダムや空港建設に対し、総力を挙げて施工体制を組む。 |
| **コスト・資源** | **リソースの効率活用・コスト削減** | **【地方での復興事業】**<br />地元の工務店と大手が組み、移動費を抑えつつ大手の最新建機を効率よく使いコストを下げる。 |

<br />

| デメリット | 具体的なデメリット | 事例・シチュエーション |
| :--- | :--- | :--- |
| **スピード** | **意思決定の遅延** | **【緊急の設計変更】**<br />現場での即断が必要な場面でも、構成各社の本社承認や合意が必要になり工事が止まる。 |
| **組織・文化** | **企業文化の違い** | **【現場ルールの不一致】**<br />安全基準やITツールの使い方が各社で異なり、現場の作業員が混乱したり対立が起きたりする。 |
| **連帯責任** | **利益と責任が連帯する** | **【パートナー企業のミス】**<br />他社が担当した箇所の施工不良でも、JV全体として発注者へ巨額の賠償責任を負う。 |
| **不透明性** | **責任の所在が曖昧になる可能性** | **【トラブル発生時の押し付け合い】**<br />共有スペースでの事故や不具合に対し、「自社の担当ではない」と責任を回避し対応が遅れる。 |

<br />

:::note  
JVによって具体的には以下のようなものが建築されています。

**歴史的・国家的シンボル**
日本の象徴となるプロジェクトでは、技術の継承とリスク分散のためにJVが組まれます。

| 構造物名 | JV活用の背景・特徴 | 主な技術的ポイント |
| :--- | :--- | :--- |
| **東京駅丸の内駅舎** | 歴史的景観の維持と現代の安全基準の両立。 | 巨大な免震構造を地下に組み込む難工事の完遂。 |
| **新国立競技場** | 日本を代表するスタジアムとしての高い公共性。 | 厳しい工期内での大規模スタジアム建設。 |
| **明石海峡大橋** | 世界最大級の吊り橋建設に伴う膨大なリスク。 | 高度な海洋土木技術による橋脚設置とケーブル架設。 |
| **皇居（新宮殿など）** | 日本の伝統技術と現代建築の高度な融合。 | 老舗ゼネコンが協力し、最高水準の品質を追求。 |

<br />

**都市開発・超高層ビル**
巨大な街づくりやランドマーク建設では、工区ごとの分担や特定技術の集約が重要です。

| 構造物名 | JV活用の背景・特徴 | 主な技術的ポイント |
| :--- | :--- | :--- |
| **東京スカイツリー** | 未踏の高さ（634m）への挑戦。 | 世界最高水準の精度管理と複雑な鉄骨構造。 |
| **六本木・麻布台ヒルズ** | 広大なエリアを一つの「街」として再開発。 | 複数の工区を異なるJVが担当し、全体を統合。 |
| **あべのハルカス** | 日本屈指の高さを誇る複合ビルの建設。 | 超高層建築における垂直搬送や風揺れ対策。 |

<br />

**社会インフラ・産業施設**  
生活と経済の基盤を支える施設は、スピードと専門性が求められる分野です。

| 分野・代表例 | JV活用の背景・特徴 | 主な実績・事例 |
| :--- | :--- | :--- |
| **トンネル・新幹線** | 長距離かつ悪地盤での掘削リスクの分担。 | 九州新幹線平間トンネルなど。 |
| **巨大ダム** | 膨大なコンクリート打設と長期管理。 | 川俣ダム、フーバーダムなど。 |
| **大型工場・プラント** | 短工期での迅速な体制構築と環境制御。 | TSMC熊本工場などの巨大半導体プラント。 |
| **公共再開発** | 自治体庁舎の建て替えや歴史的建造物の活用。 | 青森市・八代市庁舎、横浜赤レンガ倉庫など。 |

## ゼネコン以外の企業分類

---

### ハウスメーカー
ブランドイメージが確立されており、広範囲なアフターサービスや高い設計自由度が期待されることが多く、全国規模で展開している戸建て住宅系の会社で、戸建てに限らずアパートなどの低層集合住宅なども手掛けています。  
主にユーザーが所有する土地（あるいは別途探してきた土地）に対して建物を建てる「注文住宅」が主力です 。
限られた都道府県で広域に展開する中規模の工務店などは「ハウスビルダー」などと分けて呼ばれてます。  

【例】大和ハウス(株)、積水ハウス(株)、住友林業(株)、旭化成ホームズ(株)、ミサワホーム(株)など

<br />

:::tip
**アパートとマンション**

| 項目 | アパート | マンション |
| :--- | :--- | :--- |
| **主な規模** | 低層（2〜3階建てが多い） | 中層〜超高層（タワー等） |
| **主な構造** | 木造、軽量鉄骨造 | 鉄筋コンクリート造（RC）など |
| **主な建設会社** | ハウスメーカー、ハウスビルダー | ゼネコン |
| **主な企画者** | 個人オーナー、ハウスメーカー | デベロッパー（都市開発など） |

:::

### パワービルダー  
効率化を追求し、低価格な建売住宅を大量に供給する住宅会社で、床面積30坪程度の土地付き一戸建てを低価格(2,000～4,000万円程度)で販売する建売住宅会社のことです。  
基本的に「土地＋建物」をセットにした「分譲（建売）住宅」の販売であり、会社側があらかじめ土地を仕入れ、そこに自社仕様の建物を建ててから販売するため、ユーザーは土地と建物を同時に購入することになります。  

【例】タマホーム(株)、(株)オープンハウス、(株)アイダ設計など

### デベロッパー  
大規模な住宅開発や都市再開発、リゾート開発などを手掛ける会社で、施工会社ではなく建築主(施主)側の会社です。
デベロッパーの業務は、建物が建つずっと前の「企画」段階から始まります。詳細な仕事内容は以下になります。

【例】三井不動産(株)、三菱地所(株)、住友不動産(株)など

**土地の取得と企画**:膨大な土地を仕入れ、そこにどのような街やビルを作るかのコンセプトを決定します。   
**行政との折衝**: 都市計画に基づいて、道路の整備や用途地域の調整など、行政と協議を行います。  
**施工会社への発注**: 計画を実行するために、ゼネコンに対して工事を発注し、プロジェクト全体の進捗を管理します。  
**テナント誘致・販売**: 完成したビルに企業を誘致したり、マンションとして販売・管理したりします。
