# GAS 入門課題

### 課題 1: フォーム回答データの自動集計と条件付き入力（トリガーとセル操作）

**目的:** `onFormSubmit` トリガーと基本的なセルデータの取得・書き込みを習得し、自動判定ロジックを構築する。

#### 課題内容
1.  **Googleフォーム**を作成し、回答を格納する**スプレッドシート**を用意する。
2.  フォームが送信されたら自動的に実行される**インストール可能なトリガー**（`onFormSubmit`）を設定する。
3.  トリガーによって起動された関数で、以下の処理を実行する。
    * 最新の回答行のデータ（例: 点数）を取得する。
    * その点数が**70点未満**だった場合、その行の横のセルに「要指導」という文字を自動で書き込む。
    * さらに、点数の合計が格納されている**別のセル**（例: H1セル）を読み込み、最新の点数を加算して合計値を更新する。

### 課題 2: データクレンジングとシート間の移動（高度なスプレッドシート操作）

**目的:** 複数シート間のデータ移動、条件による行の削除、およびデータ構造を操作する実務的なスキルを習得する。

#### 課題内容
1.  「**未処理データ**」と「**完了データ**」という2つのシートがあるスプレッドシートを作成する。
2.  関数 `dataCleanup()` を作成する。
3.  この関数内で、以下の処理を順に行う。
    * 「未処理データ」シート全体を読み込む。
    * データ内の日付列を参照し、**1週間以上前の古いデータ**を持つ行をシートから**完全に削除**する。
    * ステータス列が「完了」となっている行をすべて見つけ出し、その行のデータを**切り取って**「完了データ」シートの最終行に**貼り付ける**。
    * すべての処理が完了したら、処理件数（移動した行の数）をGASのログに表示する。

### 課題 3: 毎日の定型業務準備とカスタムメニューの作成（時限式トリガーとUI）

**目的:** ユーザーインターフェース（カスタムメニュー）と時限式トリガーを理解し、日常業務をサポートする機能を作成する。

#### 課題内容
1.  スプレッドシートを開いたときに自動で実行される`onOpen()`関数で、シート上部に「**業務メニュー**」という**カスタムメニュー**を追加する。
2.  そのメニューから実行される関数 `createTodaySheet()` を作成する。
3.  `createTodaySheet()` は、**本日の日付名**（例: `YYYY-MM-DD`）の新しいシートを作成し、A1セルに「本日のタスク」という見出しを書き込む。
4.  GASのエディタで**時限式トリガー**を設定する。
    * このトリガーは、**毎日午前9時**に自動で実行されるように設定する。
    * 実行される関数は、特定の「設定」シートにあるセル（例: B1）に、**トリガーが実行された日時**を記録する。
