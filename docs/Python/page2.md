# ライブラリ

ここでは、Pythonにおけるライブラリについて解説します。  
ライブラリを利用する場合、個人の技術ブログが参考になることはありますが、  
記載されている情報が古い、解説しているバージョンが異なるといったことに注意してください。  

ライブラリの公式リファレンスが存在する場合はそちらを積極的に活用しましょう。  

また、標準ライブラリについては以下の公式リファレンスを活用しましょう。
