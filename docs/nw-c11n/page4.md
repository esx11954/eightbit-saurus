# 3. IPv4アドレッシング

<details>
    <summary>概要</summary>
    <div>

## IPv4アドレス

  * **表記：** `192.168.1.10` のように、ドット（`.`）で区切ります。
  * **範囲：** それぞれの数字は `0` ～ `255` までしか使えません。
  * **正体：** コンピュータ内部では「0と1の羅列（32ビット）」ですが人間には読めないため、10進数に直して表記しています。

-----

### ネットワーク部とホスト部

IPアドレスは、ただの数字の羅列ではなく、**2つの意味**を持っています。

| 用語 | 役割 | 
| :--- | :--- |
| **ネットワーク部** | **「所属するグループ」を表す** |
| **ホスト部** | **「個別の機器」を表す** | 

:::tip ポイント
同じネットワーク（同じLAN）の中にいる機器同士は、**ネットワーク部が同じ**でなければ通信できません。
:::
-----

### サブネットマスク（Subnet Mask）

**IPアドレスのどこからどこまでがネットワーク部なのか**
その境界線をコンピュータが認識するための数値です。

よく見る `255.255.255.0` という数字には、以下のような意味があります。

  * **`255` の部分：** 「ここはネットワーク部です（変えてはいけません）」
  * **`0` の部分：** 「ここはホスト部です（好きな番号を割り当ててOK）」

-----

### グローバルIP と プライベートIP

IPアドレスには、使える「場所」によって2つの種類があります。

| 項目 | グローバルIPアドレス | プライベートIPアドレス |
| :--- | :--- | :--- |
| **使われる場所** | **インターネット(WAN)** | **社内ネットワーク・家庭内(LAN)** |
| **特徴** | **世界で唯一**で、重複してはいけない。 | **組織内で唯一ならOK**で、他の会社や家と同じ番号を使っていても問題ない。 |
| **入手方法** | プロバイダ（ISP）から割り当てられる。（有料・数に限りがある） | 自由に設定できる。（無料・いくらでも使える） |
| **よく見る数字** | 多種多様 | `192.168.xx.xx`<br />`172.16.xx.xx`<br />`10.xx.xx.xx` |

:::tip 分ける必要性
世界中のすべての機器にユニークな住所（グローバルIP）を割り当てると、数が足りなくなってしまうからです。  
そこで家や会社の中では「プライベートIP」を使い、外に出る時だけルーターが「グローバルIP」に変換して通信しています（この変換機能を**NAT**と言います）。
:::
