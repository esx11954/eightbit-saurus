# 1. ネットワーク基礎

<details>

    <div>

## LAN（Local Area Network）
家やオフィスの中など、限られた範囲のネットワーク。  
インターネットという「外の世界」に出る前の、自分たちの「内側の世界」のことです。

### 主な構成要素
- 有線LAN（イーサネット）： ケーブル（LANケーブル）を物理的に挿して繋ぐ方式。安定して速い。
- 無線LAN（Wi-Fi）： 電波を使って繋ぐ方式。ケーブルがなく便利。
- ハブ / スイッチ： パソコンやプリンターなど、複数の機器を束ねる「コンセントのタコ足配線」のような役割を持つ中継器。

:::tip ポイント
LANの中にいる機器同士（例：自分のPCと隣の席の人のPC、またはオフィスのプリンター）は、外のインターネットを通らずに直接やり取りができます。
:::

## TCP/IP
世界標準の通信ルール（プロトコル）のセット。  
メーカーやOS（WindowsやMac）が違っても通信ができるのは、全員がこの「TCP/IP」という共通のルールを守っているからです。

### 2つの要素
実際は多くのプロトコルの集まりですが、代表的な2つが名前の由来です。

#### IP（Internet Protocol）
- 役割： 「宛先（住所）まで届ける」
- キーワード： IPアドレス（インターネット上の住所）
- イメージ： 荷物に「宛名ラベル」を貼って、正しい家に届ける配送ルートを決める役割。

#### TCP（Transmission Control Protocol）
- 役割： 「正確に届いたか確認する」
- キーワード： 信頼性、確認応答
- イメージ： 荷物が壊れていないか確認し、届いたら受取サインをもらう役割。  
もし途中で荷物が消えたら、再送を依頼します。
